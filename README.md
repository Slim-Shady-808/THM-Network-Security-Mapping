# THM-Network-Security-Nmap
Based off the TryHackMe Network Security Module

## Overview
- Live Host Discovery: Determine which hosts are online with ARP, ICMP, TCP, and UDP see [Live host file](01-live-host-discovery.md)
- Basic Port Scans: Identify open, closed, and filtered ports with TCP Connect, TCP SYN, and UDP scans
- Advanced Port Scans: Bypass firewalls with Null, FIN, Xmas, ACK, Window, and Idle/Zombie Scans
